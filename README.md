# Desktop Maps

This repo includes the WPF and Winforms samples using  ThinkGeo Map Control: 

- `/Samples`: A collection of feature-by-feature samples.  We suggest you start with the [How Do I Sample](https://gitlab.com/thinkgeo/public/thinkgeo-desktop-maps/-/tree/master/samples/wpf/HowDoISample) as it shows over 100  features in one easy-to-navigate app.

- `/QuickStartGuides`: Quick start guides to show you how to quickly get up and running. 

    - [Display a map in a WPF project using VS Code (.NET 7)](#repository-layout)
    - [Display a map in a WPF project using Visual Studio (.NET Framework 4.8 and .NET 7)](#repository-layout)
    - [ Display a map in a Winforms project using Visual Studio (.NET Framework 4.8 and .NET 7)](#repository-layout)

- `/Docs`: An offline version of the API documentation HTML pages.
